## 3. テストデータの商品構成

各カテゴリにおいて、以下の価格帯の商品が1点ずつ（計3点）登録されています。
プランナーはこれらを組み合わせてプランを作成します。

- **安価 (Low)**: カテゴリ内の最低価格商品 -> 「コスパ重視」に採用
- **中価格 (Mid)**: カテゴリ内の価格順中央の商品 -> 「バランス」に採用
- **高価格 (High)**: カテゴリ内の最高価格商品 -> 「ハイエンド」に採用（予算内であれば）
