# スマート購入プランナー 機能改修報告書 & ユーザーマニュアル

## 1. 変更内容まとめ

### ■ 変更前の仕様と課題
- **検索の信頼性不足**: ユーザーが入力した意図（目的）とは無関係な商品が表示されることがありました。「asdf」などの無意味な入力に対しても何らかの商品が提案されてしまう状態でした。
- **提案内容の固定化**: 「最安」か「最新」のどちらかしか選択肢がなく、予算を多めに設定しても低価格な商品ばかりが選ばれるなど、予算に応じた柔軟な提案ができていませんでした。
- **視覚情報の欠如**: 提案結果に画像が表示されず、どのようなセットなのかを直感的に判断しにくい状態でした。

### ■ 変更後の仕様
キーワードから商品カテゴリを厳密に特定し、目的と予算に合わせた「3つのコンセプト別プラン」を提案する仕様に刷新しました。

1. **厳格なキーワードマッピング**:
   - `planner_keyword_categories` テーブルを導入。ユーザーの目的語（ゲーム、食料品等）を正確に内部カテゴリへ変換します。
   - 登録されていないキーワードや、2文字未満の短い入力、意味のない文字列（test, aaaa等）はバリデーションで弾き、精度の低い提案を停止しました。
2. **外部画像URL（image_url）のサポート**:
   - `items` テーブルに `image_url` カラムを追加。Unsplash等の高品質な外部フリー素材を表示可能にしました。
   - ローカル画像と外部URLのハイブリッド表示に対応し、リッチなUIを実現しました。
3. **3プラン生成ロジック**:
   - **コスパ重視プラン**: とにかく安く揃える。各カテゴリの最安値を選択。
   - **バランスプラン**: 質と価格の両立。価格帯の中央値付近から選択。
   - **ハイエンドプラン**: 予算上限を最大限活用。最高価格の商品を選び、予算オーバー時のみ順次グレードを下げる高度な調整ロジックを搭載。

## 2. 使い方（ユーザー向けマニュアル）

### ■ 基本的な操作方法
1. **メニュー選択**: ナビゲーションバーの「プランナー」をクリックします。
2. **目的と予算の入力**:
   - **利用目的**: 下記の有効キーワード（またはそれを含む文章）を入力します。
   - **予算**: プランの合計金額の上限を選択します。
3. **プランの比較**: 提示された3つのプランから、自分の好みに合うものを選択し、商品名をクリックして詳細を確認できます。

### ■ 有効なキーワード例
以下のキーワードを入力することで、適切なセットが提案されます。

| カテゴリ | 入力キーワード例 |
| :--- | :--- |
| **デジタル** | ゲーム、プログラミング、動画編集、在宅ワーク、家電 |
| **ライフスタイル**| 自炊、パーティー、ダイエット、食料品、生活、服 |
| **学習・教養** | 読書、勉強、書籍 |

### ■ 表示内容の解説
- **プラン詳細**: 商品ごとの画像、名称、価格、カテゴリが表示されます。
- **合計金額**: そのプランで全商品を揃えた際の税込総額です。
- **残予算**: あなたが設定した予算から、どれだけお釣りがくるかを表示します。

### ■ 注意点
- キーワードが含まれない入力（例：「こんにちは」）では、商品は表示されません。
- 予算が極端に低い場合、セット（複数商品）を組むことができず、結果が空になることがあります。
- 本機能はデータベース内の在庫状況に依存します。
